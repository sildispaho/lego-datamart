# lego-datamart
ETL pipeline and SQL Server schema for loading and analyzing LEGO dataset.
This repository contains the SQL Server schema.

/*FILES*/

- db_structure.sql --> Contains the schema of source DB. 
- indexes.sql --> Contains indexes of the tables
- Screenshot db_diagram --> A database diagram schema of what we have in sql Server.

/*HOW TO USE */

Run db db_structure.sql in SSMS, after that indexes.sql
